## Mockups
Se plantearon los mockups con el objetivo de ofrecer una representación fácil de entender para el cliente, organizado respecto a sus funciones principales.
### Mockup - Pantalla Principal
La pantalla principal reúne las funciones esenciales del taller y organiza la navegación de forma clara y directa, priorizando el acceso rápido a cada módulo.

### Mockup - Gestión de Productos
La sección de gestión de productos permite identificar con facilidad cada artículo mediante su nombre y datos clave como stock disponible y stock actual, lo que agiliza la reposición. Incluye una barra de búsqueda que admite filtros por distintos parámetros para localizar productos de forma precisa.
<img width="1920" height="1080" alt="Mockup - Gestión de Productos" src="https://github.com/user-attachments/assets/027e7eb1-de77-4cd5-9b06-764574b40129" />
### Mockup - Registro de Ventas
El módulo de registro de ventas presenta un historial ordenado donde se muestran primero las transacciones más recientes. Destaca información fundamental como fecha y monto, además de reportes relevantes que incluyen margen de ganancia y productos con mayor rotación.
