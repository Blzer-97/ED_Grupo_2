### Requisitos de usuario
El sistema debe permitir que el usuario registre nuevos productos ingresando su nombre, SKU, costos, precios, stock y características. También debe facilitar el registro de compras y ventas, ajustando el stock según los movimientos y permitiendo actualizarlo cuando ingresen nuevas existencias. Para cada venta, el sistema debe calcular automáticamente los importes correspondientes, incluido el IGV y el total a pagar. Debe generar una boleta que el cliente podrá luego validar en su propio sistema vinculado a SUNAT. El usuario podrá consultar reportes de ventas por periodo, estimar márgenes de ganancia dentro del rango indicado y revisar productos con mayor o menor movimiento. Asimismo, el sistema debe gestionar servicios, permitiendo registrarlos, listarlos para su uso en la boleta y mostrar su margen de ganancia.
