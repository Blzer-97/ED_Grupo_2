## Sistema de Gestión de Inventario y Ventas para un Taller Automotriz

El presente plan de proyecto tiene como propósito describir el desarrollo del software “Sistema de Gestión para una Mecánica Automotriz” destinado a un cliente real. Su objetivo principal es diseñar y construir una solución que permita administrar de manera eficiente las operaciones de un taller mecánico, incluyendo el registro de ingreso de stock y ventas, la gestión de servicios ofrecidos, el control de inventario junto a su actualización automática y la generación de reportes administrativos. La meta es optimizar los procesos internos del taller, reemplazando su registro manual con Excel del cliente por una herramienta más centralizada y automatizada.

El tiempo de ejecución proyectado contempla las etapas de análisis, diseño, desarrollo, pruebas y despliegue del prototipo funcional. Entre las limitaciones identificadas se consideran la disponibilidad del personal del taller para entrevistas y pruebas, la posible falta de experiencia del equipo en las herramientas de desarrollo seleccionadas y el tiempo limitado para implementar todas las funcionalidades previstas en la primera versión del sistema.
